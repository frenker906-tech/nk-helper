
Привет! 👋 Это обновленный NK Helper — твой универсальный мультитул для диагностики, восстановления системы и борьбы с вредоносным ПО!
NK Helper — это портативное приложение, объединяющее в себе все самые необходимые системные утилиты, инструменты разблокировки и функции антивирусного сканирования. Интерфейс выполнен в современном темном стиле, работает быстро и предоставляет доступ к критически важным настройкам Windows.

✨ Основной функционал (Навигация по вкладкам)

🏠 Главная панель (Dashboard)
Стартовая страница, с которой начинается работа:

Быстрый запуск: Доступ к браузерам, Блокноту и Панели управления в один клик.

Мониторинг: Отображение текущей нагрузки на ПК (ЦП, ОЗУ, Свободное место на диске).

Среда выполнения: Выбор целевого системного диска (критически важно при работе из WinRE/LiveUSB) и просмотр базовой информации о системе.

Свое окно "Выполнить": Запуск любых системных команд напрямую.

📁 Файловый менеджер
Встроенный двухпанельный (визуально) проводник, который работает даже если explorer.exe "упал":

Поддержка отображения скрытых и системных файлов.

Базовые операции: Копирование, Вырезание, Вставка, Удаление.

Встроенный "NK Блокнот": Позволяет читать и редактировать текстовые и конфигурационные файлы прямо из утилиты.

💽 Менеджер дисков
Инструмент для управления логическими томами:

Просмотр списка всех дисков, файловой системы и свободного места.

Опасная зона: Возможность принудительно отмонтировать/удалить том из системы (скрыть букву диска).

📊 Диспетчер задач
Мощная альтернатива стандартному Task Manager:

Просмотр активных процессов (PID, потребление памяти).

Завершение (Kill) и Заморозка/Разморозка (Suspend/Resume) процессов.

Снятие статуса "Критический": Уникальная функция, позволяющая снимать защиту с системных или вирусных процессов, чтобы их можно было безопасно закрыть без синего экрана (BSOD).

Быстрый переход к расположению исполняемого файла.

⚙️ Службы
Удобный менеджер системных служб Windows (чтение статуса, запуск и остановка).

📝 Редактор реестра
Полноценный regedit внутри программы:

Навигация по всем основным веткам (HKCU, HKLM, HKCR, HKU, HKCC).

Чтение, изменение и удаление строковых и числовых параметров реестра.

💻 Терминал (CMD)
Встроенная консоль прямо в интерфейсе программы. Поддерживает историю директорий (cd) и выполняет стандартные команды Windows в фоновом режиме, выводя результат на экран.

🚀 Расширенная Автозагрузка
Продвинутый сканер автозагрузки, который ищет "хвосты" там, куда обычные юзеры не смотрят:

Сканирование стандартных веток Run (HKCU, HKLM, WOW6432Node).

Проверка скрытых способов запуска: AppInit_DLLs, Shell/Userinit, Setup.

Сканирование папок Startup всех пользователей.

Возможность редактировать и удалять найденные записи в один клик.

👥 Пользователи
Инструменты для управления учетными записями локального ПК:

Смена пароля любого пользователя (без знания старого).

Включение/отключение учетных записей.

Создать Админа: Моментальное создание скрытой учетной записи NKAdmin с правами администратора (бэкдор для восстановления доступа).

🛠️ Восстановление & Фиксы (NK Anti-Malware)
Самый мощный раздел утилиты, разделенный на три категории:

Антивирусные утилиты: * Массовое убийство всех подозрительных процессов, запущенных из папок Temp и AppData.

Сброс Proxy-сервера (если вирус заблокировал доступ в интернет).

Снятие атрибутов "Скрытый/Системный" со всего диска (лечение последствий вирусов на флешках/дисках).

Разблокировка системы:

Снятие всех ограничений групповых политик (разблокировка Диспетчера задач, CMD, Реестра).

Очистка IFEO-дебаггеров (фикс, когда вирусы не дают запустить антивирусные утилиты).

Запуск проверок целостности системных файлов (sfc /scannow и DISM).

Сброс сети (DNS/Winsock), очистка кэшей Temp/Prefetch и восстановление стандартной темы/шрифтов.

Системные твики и Защита:

Подмена utilman.exe: Классический трюк для получения доступа к ПК с экрана блокировки. Заменяет системную утилиту на сам NK Helper.

Сброс файла hosts.

Включение/отключение UAC (Контроля учетных записей).

Клонирование (Защита): Программа прячет свою копию со случайным именем в системных директориях, чтобы вирус не мог ее удалить.

🕒 Недавние файлы
Инструмент для поиска свежих вирусных дропперов:

Поиск недавно созданных или измененных файлов (за последние 7 дней).

Доступен быстрый поиск (только по папкам Users, Temp, ProgramData) и глубокий поиск по всему диску.

🌐 Сетевой монитор
Отображение всех активных сетевых соединений (TCP/UDP) в реальном времени, привязка их к PID процессов и возможность моментально убить процесс, отправляющий подозрительный трафик.

⌨️ Вспомогательные инструменты (Боковое меню)
Проводник Windows: Запуск нативного explorer.exe прямо из утилиты.

Виртуальная клавиатура: Кастомная экранная клавиатура на случай, если драйверы физической клавиатуры "слетели" или заблокированы.

⚠️ Предупреждение: Утилита обладает мощным функционалом, способным вмешиваться в критически важные системные процессы и файлы реестра. Рекомендуется запускать программу с правами Администратора для полноценной работы всех функций (включая работу с реестром, службами и системными каталогами). Используйте инструменты вкладки "Менеджер дисков" и "Фиксы" с осторожностью!
